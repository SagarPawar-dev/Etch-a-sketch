# 🎨 Etch A Sketch

A browser-based drawing app inspired by the classic Etch A Sketch toy — built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://sagarpawar-dev.github.io/Etch-a-sketch/

---

## Preview



![Etch A Sketch Demo](/gif/demo.gif)



---

## Features

- 🖱️ Draw by clicking and dragging the mouse
- 👆 Touch support for mobile devices
- 🎨 Color picker to choose any color
- 🌈 Random color mode for rainbow drawing
- 🧹 Eraser tool
- 🗑️ Clear grid button
- 📐 Adjustable grid size (1x1 to 100x100)
- 📱 Responsive design for mobile and desktop

---

## How To Use

1. Open the live demo link
2. Click **Create Grid** to generate the drawing pad
3. Use the **Color Picker** to select a color
4. Click and drag on the grid to draw
5. Switch to **Random** mode for rainbow colors
6. Use **Eraser** to erase specific areas
7. Use **Clear Grid** to start fresh
8. Adjust the slider to change grid size
